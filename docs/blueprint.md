# **App Name**: StyleSavvy AI

## Core Features:

- User Profiling: Collect data about the user's body type, skin tone, and preferred styles via a questionnaire and/or image analysis.
- Outfit Recommendation: Based on user profile data, use a generative AI tool to suggest outfits that complement their features. AI takes into account color theory, current trends, and individual preferences.
- Shopping Platform Integration: Connect with popular online shopping platforms to source the recommended clothing items.
- Visualizer: Display the suggested outfits on a virtual model or allow users to upload their own photos for a personalized view.
- Style Guide: Provide a detailed style guide explaining why the suggested outfits work well for the user's features, including color palettes, cuts, and materials.
- Outfit Saving: Enable users to save their favorite outfits for future reference.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082). Evokes sophistication and style, drawing from concepts of fashion and art.
- Background color: Very light grey (#F5F5F5). Complements the deep indigo without overpowering the visual elements.
- Accent color: Violet (#8F00FF). Used to highlight interactive elements and calls to action, creating contrast and visual interest.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'PT Sans' (sans-serif) for body text.
- Use fashion-related icons such as clothing, shoes, accessories to help showcase available clothes
- Clean, modern layout with ample white space for a premium feel.
- Smooth transitions and subtle animations when navigating through outfit recommendations and shopping options.
